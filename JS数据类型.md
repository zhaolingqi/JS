开始给变量赋值为null，一般表明该变量赋值为一个对象

使用过程中赋值为null， 使其指向的的对象被回收

#### IIFE（Immediately-Invoked Function Expression） 匿名函数自调用

##### 作用

隐藏实现，不会污染外部（全局）命名空间

用于编写js模块

```javascript
(function() { //匿名函数自调用
 var a = 3
 console.log(a + 3)
 })()
```
